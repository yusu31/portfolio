# セクション仕様書 — yusu portfolio v2

## 概要

各セクションはUIレイヤー（HTML）とCanvasレイヤー（3D）が連動する。
スクロール進行度（0→1）に応じてカメラとオブジェクトが変化する。

---

## カメラパス設計

```
scroll: 0.0  → Hero（クリスタルを正面から見る）
scroll: 0.15 → Impact（クリスタルが下がり、数字テキストが前景に）
scroll: 0.3  → Story（カメラが左旋回、背景が暗くなる）
scroll: 0.5  → Projects（クリスタルが展開・爆発エフェクト）
scroll: 0.7  → Skills（パーティクル密度が上がる）
scroll: 0.85 → Blog（ミニマル、パーティクルのみ）
scroll: 1.0  → Contact（暗転、フォームが前面に）
```

---

## 各セクション仕様

### Hero
| 項目 | 内容 |
|---|---|
| UIレイヤー | バッジ、HEY.テキスト、サブコピー、CTAボタン2つ |
| Canvasレイヤー | Three.js クリスタル（Icosahedron）+ 3点ライティング |
| インタラクション | マウスパラックスでクリスタルが追従回転 |
| アニメーション | GSAP SplitText で "HEY." が下から入場 |
| スクロール変化 | クリスタルが徐々に縮小・遠ざかる |

---

### Impact（数字で見る）
| 項目 | 内容 |
|---|---|
| UIレイヤー | 4つの数字カード（10年 / 3プロジェクト / 80% / 週2h） |
| Canvasレイヤー | パーティクルが疎らに浮遊 |
| アニメーション | 数字のカウントアップ（GSAP） |
| スクロール変化 | パーティクルがゆっくり上昇 |

---

### Story

|---|---|

| スクロール変化 | カメラが緩やかに左旋回 |

## モバイル対応方針

| 要素 | PC | SP（≦768px） |
|---|---|---|
| Canvas | 全画面表示 | 非表示（CSS background-imageで代替） |
| カーソル | 球体カーソル | 非表示（標準カーソル） |
| アニメーション | フル | `prefers-reduced-motion`準拠で最小化 |
| レイアウト | 2カラム | 1カラム |
